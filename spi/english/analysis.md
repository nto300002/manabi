# 英語学習：3つの柱に基づく分析と対策

## 🏛 1. 統語論（文の骨組み）

- **傾向**: SVOC構造の理解は進んでいるが、形式目的語（it）のスロットや、構文変換（SVO CからSVCへの書き換え）の際にミスが生じやすい。
- **対策**: 「make it C to do」などの特定スロットのパターン化を徹底する。

## ⏱ 2. 時制論（時間の論理）

- **傾向**: 主節の動詞（アンカー）と時制を同期させる「時制の一致」が最大の課題。仮定法（were）や完了形の継続（for）の選択で、アンカーを見失う傾向がある。
- **対策**: V-Anchor（動詞の錨）固定プロトコルを適用。問題を解く前に、主節の時制フラグ（was, did等）を脳内で固定してから選択肢を絞り込む。

## 🔗 3. 意味論（概念の相関）

- **傾向**: 単発の語彙力は高いが、多義語のコアイメージの把握や、SPI特有のアナロジー（包含関係・機能的類似）の強化が必要。
- **対策**: 多義語（address, apply等）のコアイメージを体系化し、単語間の「因果関係」を意識して学習する。

## ⚠️ 要注意パターン

- **助動詞**: `had better` + 原形（shouldとの混同に注意）
- **仮定法**: `If I were`（主語に関わらずwere）
- **期間と起点**: `for` + 期間 / `since` + 起点
- **時制一致**: 文全体の時制 OS（過去/現在）を最初に確認する。
